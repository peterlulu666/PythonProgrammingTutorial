### Turtle graphics

Moving a turtle

    ```python
    import turtle
    aScreen = turtle.Screen()
    shelly = turtle.Turtle()
    shelly.forward(100)
    shelly.right(90)
    ```
    
Color      

    ```python
    import turtle
    shelly = turtle.Turtle
    shelly.color('blue')
    shelly.width(10)
    ```
